Pt. 1: Selecting samples for building bait panel

This dataset has many samples that failed in library prep or sequencing. We want to use high quality samples for building the bait panel so our selection of loci isn’t biased by large amounts of missing data. To select those samples, we can look at how many SNPs were able to be genotyped in each individual and what the read depth was at the SNPs that were sequenced. Below are histograms for those two factors with the line indicating the cutoff that I used for inclusion in the set of samples used to build the bait panel. This should provide a set of individuals with relatively complete data with relatively high confidence genotype calls.

dat %>% 
  filter(propMissing < 0.45) %>% 
  filter(meanDepth > 5) %>% 
  filter(maf > 0.02) %>% 
  filter(dScore > -7 | dScore < 7) %>% 
  filter(allelicBalance > 0.3 | allelicBalance < 0.7) %>% 
  filter(snps < 10) %>% 
  filter(hetPerc < 0.6) %>%
